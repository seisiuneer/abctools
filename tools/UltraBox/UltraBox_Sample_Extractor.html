UltraBox Sample Extractor

Extract samples from .sf2 soundfont files, preview them, and export loop-aware WAV files.

This utility helps you extract sounds from .sf2 soundfont files, and save them as individual .wav files.

To start, drop a file in one of the supported formats on this page, or open a file from your system:

      Supported Formats

      • .sf2
      • .js: UltraBox built-in sample script
        Should also work with the sample scripts from Pandora's Box
      • .wav: Loadable for convenience, in case resampling is needed. Loop points are also saved/loaded, so they can be consulted after exporting.
      • .mp3: Resampled to 44100hz on load. No loop point info.
      • .ogg: Resampled to 44100hz on load. No loop point info.
      • .flac: Resampled to 44100hz on load. No loop point info.
