Ocarina Tabs
============

12 hole ocarina fingering font and [tab creator](http://panzi.github.io/ocarina_tabs).

Example tab: [Clocktown Theme](http://tinyurl.com/clocktown) from The Legend of Zelda: Majoras Mask
