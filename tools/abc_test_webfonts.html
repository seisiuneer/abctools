Test Webfonts for ABC Compatibility

On Chrome, to read out all the available font family names, click Get all font family names from Chrome.

You may be prompted by Chrome to give permission for the operation.

Alternatively, for all browsers click Open an existing abc_test_webfonts full font family names file to load a previously saved full font family list saved using Chrome from this utility.

The All Full and Postscript Font Family Name Pairs area will be filled with the names of all fonts either obtained from Chrome or loaded from a file.

Both the full and Postscript font family name for each font are displayed.

The ABC-Compatible Font Family Names will be filled with a list of the font family names that can be used in ABC font directives.

Click Copy Full and Postscript font family names to the clipboard to copy the full list of font family names to the system clipboard for pasting into other applications.

Click Save Full and Postscript font family names to a file to save the full list of font family names to a file.

Click Copy ABC-compatible font family names to the clipboard to copy the list of ABC-compatible font family names to the system clipboard for pasting into other applications.

Click Save ABC-compatible font family names to a file to save the list of ABC-compatible font family names to a file.

The ABC-compatible font family names can be used in %%titlefont and other ABC font directives in the ABC Transcription Tools.

You can preview the fonts in referenced by their full name at Wordmark.
