Custom Instrument Range and Tuning:

The custom Northumbrian Pipes instruments created by this tool are natively in the key of F.
They are designed to be played using Tin Whistle/Flute or Uilleann Pipes fingering patterns on the WARBL.

Playable Range (WARBL / MIDI input)
B3 (59) → D6 (86)

Sounding Pitch (what you hear)
D4 → F6 (two octaves above)

Drones
The drones sound at Eb3 (MIDI note 51).
The lowest chanter note that matches the drones is D4 (MIDI note 62).

Transposition
The instrument can be transposed to any key using Transpose (semitones) in Celtic Sounds
