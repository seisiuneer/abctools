Instructions
Getting started
1) Click Connect MIDI and if requested, allow browser MIDI access.

2) Choose your MIDI Input and Input Channel.

3) Select an Instrument and the Expression Source you want the sound module to follow.

4) Adjust volume, attack, release, pitch bend scaling, reverb, and transpose as needed.

5) Play your controller and watch the Live Status and optional MIDI Message Stream.
Expression source
Volume (CC7) follows MIDI CC7.
Breath (CC2) follows MIDI CC2.
Expression (CC11) follows MIDI CC11.
Note Velocity uses incoming note-on velocity.
Fixed Full Volume ignores expression and always plays at full level.
Pitch bend scaling
The Pitch Bend Up Scale and Pitch Bend Down Scale let you independently control how strongly pitch bend messages affect upward and downward bends.
Reverb
Choose a reverb style, then adjust:

Wet = effected signal level
Dry = direct signal level
Pre-delay = delay before reverb begins
Troubleshooting
If you do not hear sound:

• Make sure MIDI access was granted.
• Verify the correct MIDI device is selected.
• Check that the correct input channel is selected.
• Confirm your samples and impulse response files are present at the expected paths.
• Try clicking All Notes Off if a note gets stuck.
Browser compatibility
This tool requires a browser that supports the Web MIDI API.
If your browser does not support WebMIDI, MIDI devices will not appear and the tool will not function.
Recommended browsers:
• Google Chrome (Windows, macOS, Linux)
• Microsoft Edge (Windows, macOS)
• Opera (desktop versions)
• Chromium-based browsers that support WebMIDI
Not currently supported:
• Safari (macOS and iOS)
• Most mobile browsers
