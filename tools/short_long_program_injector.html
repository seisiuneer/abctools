ABC MIDI Short/Long Note Program Injector

Injects a configurable long-note MIDI program and a configurable short-note MIDI program based on note length and short-note run length.

Input ABC

Output ABC

Ready.
How it works: A note is considered short when its actual duration equals the configured short-note length. Any note longer than that is treated as long. Short runs only switch to the short-note program once they reach the configured number of consecutive short notes. Until then, they stay under the current long-note phrase.
