Instructions
Getting Started
  1. Click Connect MIDI and allow browser MIDI access if requested.
  2. Choose your MIDI Input and Input Channel.
  3. Select one of the five instrument voices: Air, Reed, Brass, Pad, or Lead.
  4. Choose the Expression Source that matches your WARBL setup. Most players will start with Breath (CC2).
  5. Adjust tone, brightness, attack, release, reverb, and bend scaling to taste, then play your WARBL and watch the live status and MIDI log.
Instrument Voices
Air is the cleanest and usually the best starting point for checking pitch response.
Reed emphasizes odd harmonics for a reed-like tone.
Brass adds stronger buzz and bite.
Pad uses a softer, chorused pad-like tone.
Lead is a brighter, more electronic monophonic lead.
Expression Source
Breath (CC2) follows MIDI CC2.
Expression (CC11) follows MIDI CC11.
Volume (CC7) follows MIDI CC7.
Note Velocity uses note-on velocity as the playing level.
Fixed Full Volume ignores incoming expression and plays at a constant level.
Pitch Bend and Portamento
The default Pitch Bend Range is ±2 semitones, which is the most common WARBL setup.
Bend Up Scale and Bend Down Scale let you make upward and downward bends feel different without changing the WARBL itself.
When Enable Portamento is turned on, the synth glides from one note to the next over the selected Portamento Time. It is off by default for all patches.
Reset Buttons
Reset Current Preset to Defaults restores only the currently selected voice to its default settings.
Reset All Presets to Defaults clears saved changes for all voices and restores the current voice to its default settings.
WARBL Configuration Tool
The WARBL Configuration Tool button in the upper-left corner of the page opens the official WARBL Configuration Tool in a new browser tab.
This allows you to adjust fingering patterns, pressure sensitivity, MIDI output behavior, and pitch bend settings while using WARBL Sound Lab.
The configuration tool runs separately from this synth, so you can switch between the two browser tabs as needed.
Recommended Starting Setup
For initial testing, try Air with Breath (CC2), bend range at 2 semitones, both bend scales at 1.00, and portamento turned off.
Once the basic note response feels right, add brightness-follow, reverb, and optional portamento.
Troubleshooting
If you do not hear sound:
• Make sure MIDI access was granted.
• Verify the correct MIDI device is selected.
• Check that the correct input channel is selected.
• Try clicking All Notes Off if a note gets stuck.
Browser Compatibility
This tool requires a browser that supports the Web MIDI API.
If your browser does not support WebMIDI, MIDI devices will not appear, and the tool will not function.
Recommended browsers:
• Google Chrome (Windows, macOS, Linux)
• Microsoft Edge (Windows, macOS)
• Opera (desktop versions)
• Chromium-based browsers that support WebMIDI
Not currently supported:
• Safari (macOS and iOS)
• Firefox (sometimes prompts for a WebMIDI plug-in, may or may not work)
• Web MIDI browser for iOS
• Most mobile browsers
