ABC Sustained Note Limiter

  • Paste ABC notation into the input area or open one or more .abc or .txt files.
  • Click Limit Sustained Notes. Every tune and voice is analyzed independently.
  • A tie is removed only when it carries a matching note or chord tone across a measure line and retaining the next tied segment would exceed the selected duration.
  • Tempo is read from Q:. Meter and default note length are read from M: and L:. Inline field changes are supported.
  • If no usable tempo is present, that tune is left unchanged and reported in the status.
  • The output preserves all input text except for tie hyphens that must be removed.
