ABC Transcription Tools Instructions

Getting Started:

To begin either paste some ABC into the text box or select a .ABC or .TXT file from your computer by clicking the "Select .ABC File" button.

This converter can handle single or multiple tunes in a single .ABC file

The ABC will be converted to notation and displayed below a set of control buttons:

Transpose and Clear Controls:

Transpose Down - Transposes the tune down one note letter name

Transpose Up - Transposes the tune up one note letter name

Clear - Clears the ABC text area

Advanced Controls - Clicking this shows a new panel with several additional controls:

Toggle Annotations - Shows/Hides most ABC annotations like Discography, History, etc.

Toggle Text - Shows/Hides any %%center or %%text based text in the ABC

Toggle Chords - Shows/Hides any chords entered in the ABC

Set Staff Spacing - Allows you to set a wider gap between the staffs

Notation/Tablature Display Controls:

These control how the notation is displayed.

Sheet Music - Standard notation only

Note Names Tab - Shows standard notation and note names below

Mandolin Tab - Shows standard notation and tablature for Mandolin

Guitar EADGBE - Shows standard notation and tablature for standard EADGBE-tuned guitar

Guitar DADGAD - Shows standard notation and tablature for DADGAD-tuned guitar

Whistle Tab - Shows standard notation and tablature for whistle in the key of D

Display Size Options:

These can be useful if you just want to screen capture the generated notation or review your work:

Mini

Small

Medium

Large - Perfect for printing

PDF Export:

Export PDF - Exports the currently displayed score as a high quality PDF file.

Each tune in a multi-tune ABC file will be rendered starting on a new page.

Printing:

When printing the page using your browser's Print command, only the generated notation is printed.

Using your browser's Print-to-PDF feature is an ideal way to get two tunes on a single page PDF file.

Setting the Display Size to Large and then printing from your browser can achieve some nice results.

Limitations:

Multi-tune PDF conversion is limited by your browser's system resources.

The converter should have no problem converting typical sets of 3 to 5 tunes.

Note name tablature can handle key changes only at the beginning of a staff, not in the middle.
