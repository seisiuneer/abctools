ABC Transcription Tools Website Builder

This tool generates a website from a file of ABC tunes.

At the top of the generated website is a selector for the tunes.

Below the selector is an iframe that will contain the selected tune loaded into the ABC Transcription Tools player.

Instructions:

1) Click "Open ABC File" to open a file of tunes.

2) Click "Generate Website".

The website code will be created and loaded into the Website Code area.

Click "Save Output to a file" to save the results to a file.

Click "Copy Output to the clipboard" to copy the results to the system clipboard for pasting into other applications.
