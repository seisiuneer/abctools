ABC Transcription Tools Release Notes

This ABC to notation converter was developed by Michael Eskin.

Full project source may be found at: GitHub.

Known issues and limitations:

If you aren't seeing the latest features, make sure you have the latest code by clearing your browser cache or doing a force reload on the converter page.

Safari and Mobile Safari have many rendering issues with abcjs.

This includes all iOS browsers, which are all just skins over the built-in Safari-based web control.

Desktop Chrome, Edge (Chrome-based), and Firefox work best with this converter.

Text tab notes with natural indication show the wrong note name. abcjs is giving me the wrong note number to translate to text. No fix at this time.

Transpose up and down are unable to deal with text in chord annotations and produces odd results.

Transpose up and down are done using just letter transposition with some intelligence around the octaves.

The transposer doesn't have any real idea about the key, so may not produce the expected result for some cases, for example Bmin transposed up will result in Cmin, not C#min.

Multi-tune conversion is limited by your browser's system resources.

The converter should have no problem converting typical sets of 3 to 5 tunes.
